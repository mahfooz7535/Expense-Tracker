#Expense Tracker App
A simple Expense Tacker built using HTML, CSS, and JAVASCRIPT.

#Features 
. Add Expense 
. Edit existing expenses
. Delete individual expenses 
. Automatic balance calculation 
. Data saved using Local storage 
. Clean and responsive user interface 

#Technologies used 
. HTML
. CSS
. JAVASCRIPT
. LOCALSTORAGE 

#What I Learned

By building this project, I improved my understanding of:

- JavaScript fundamentals
- DOM manipulation
- Event handling
- CRUD operations (Create, Read, Update, Delete)
- Local Storage
- Responsive web design
- Problem-solving through real-world projects

#Purpose

The main goal of this project was to apply my web development knowledge in a practical way and gain hands-on experience building an interactive web application from scratch.
